# Python-Username-Validator

A beginner Python project that validates a username based on simple rules.

## Features

* Checks if the username is **12 characters or fewer**.
* Checks if the username **does not contain spaces**.
* Checks if the username **contains only letters** (no numbers or special characters).
* Displays a welcome message if the username is valid.

## Concepts Used

* `input()`
* `if`, `elif`, `else`
* Logical operators (`not`)
* `len()`
* `find()`
* `isalpha()`
* f-strings
